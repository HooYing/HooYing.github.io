Linux磁盘扩展

1. 判断磁盘是否采用LVM管理

# 查看设备的文件系统信息
lsblk -f
# 观察输出中是否存在 lvm 类型标识。若某设备的 FSTYPE 字段显示为 LVM，则属于 LVM 逻辑卷
NAME        FSTYPE      LABEL UUID                                 MOUNTPOINT
sda                                                                
├─sda1      ext4             5a3b...                               /boot
└─sda2      LVM2_member      xyz1...                               
  ├─vg-root xfs              1234...                               /
  └─vg-swap swap             abcd...                               [SWAP]

2. 非LVM根分区扩容步骤

# 40GB扩展到80GB
# 查看根分区在哪个设备上
[root@localhost linux_client-develop]# lsblk
