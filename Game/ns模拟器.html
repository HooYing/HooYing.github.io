ns模拟器

1. 编译前提

1.1 初始化git子模块

git submodule update --init --recursive

1.2 .git/config把失效的github地址换一下

如：https://github.com/yuzu-emu/breakpad.git -> https://github.com/yuzu-emu-mirror/breakpad

https://github.com/merryhime/dynarmic.git -> https://github.com/yuzu-mirror/dynarmic.git

再执行：git submodule update --init --recursive

1.3 cmakelists.txt 把失效的github地址换一下

set(NASM_DOWNLOAD_URL "https://github.com/yuzu-emu/ext-windows-bin/raw/master/nasm/nasm-${NASM_VERSION}-win64.zip")

set(NASM_DOWNLOAD_URL "https://github.com/yuzu-mirror/ext-windows-bin/raw/master/nasm/nasm-${NASM_VERSION}-win64.zip")

file(DOWNLOAD
            https://github.com/yuzu-emu/ext-windows-bin/raw/master/clang-format${CLANG_FORMAT_POSTFIX}.exe
            "${CLANG_FORMAT}" SHOW_PROGRESS
            STATUS DOWNLOAD_SUCCESS)

file(DOWNLOAD
            https://github.com/yuzu-mirror/ext-windows-bin/raw/master/clang-format${CLANG_FORMAT_POSTFIX}.exe
            "${CLANG_FORMAT}" SHOW_PROGRESS
            STATUS DOWNLOAD_SUCCESS)

1.4 DownloadExternals.cmake把github地址换一下

set(package_base_url "https://github.com/yuzu-emu/") ->

set(package_base_url "https://raw.githubusercontent.com/yuzu-mirror/")

ext-windows-bin/raw/master/ -> ext-windows-bin/master/

2. 编译windows版本

windows编译教程：https://git.tardis.systems/mirrors/yuzu/wiki/Building-for-Windows

依赖：

  • Visual Studio 2022 Community
  • CMake
  • Vulkan SDK
cmake .. -G "Visual Studio 17 2022" -A x64
cmake --build . --config Release

3. 编译Android版本

Android编译教程：https://git.tardis.systems/mirrors/yuzu/wiki/Building-For-Android

依赖：

  • Android Studio
  • NDK 25.2.9519653 and CMake 3.22.1
  • Visual Studio 2022 Community
  • Vulkan SDK

编译：

  1. Start Android Studio, on the startup dialog select Open.
  2. Navigate to the yuzu/src/android directory and click on OK.
  3. In Build > Select Build Variant, select release or relWithDebInfo as the "Active build variant".
  4. Build the project with Build > Make Project or run it on an Android device with Run > Run 'app'.
